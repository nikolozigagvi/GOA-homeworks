About the Game

Realm of Shadows is a Windows-compatible ASCII adventure RPG written fully in Python. Explore forests, caves, and dark castles while fighting monsters, looting gold, upgrading weapons, and surviving the darkness.

Core Features

  • ✔ Turn-based combat system
  • ✔ Shops, rest points, and item upgrades
  • ✔ Different biomes with unique monsters
  • ✔ Save/load system using JSON
  • ✔ Works on all Windows terminals
  • ✔ Full ASCII retro experience
